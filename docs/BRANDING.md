# Branding Direction

## Recommended working name

**Perforce Companion**

Why:
- Clearly communicates that it complements rather than replaces P4V.
- Fits the local companion architecture.
- Works for ChatGPT users without requiring them to understand MCP.

Before public release, confirm trademark and naming permissions. Avoid implying that the product is officially produced or endorsed by Perforce Software unless written permission is obtained.

## Alternative names

- Helix Assistant — likely higher trademark risk
- Depot Companion — broader but less immediately recognisable
- P4 Companion — concise, but requires naming review
- Workspace Pilot — distinctive but less searchable

## Product descriptor

**A safe Perforce integration for ChatGPT, Unreal Engine, and UnrealGameSync teams.**

## Voice

- Calm
- Technical but understandable
- Transparent about uncertainty
- Safety-focused without sounding alarming
- Never playful during destructive-action confirmations

## Visual direction

- Clean developer-tool aesthetic
- Strong connection/status symbolism
- Avoid copying P4V or OpenAI branding
- Use a linked-node, depot-path, or branching-workspace motif
- Icon must remain recognisable at small plugin-directory sizes

## Colour direction

Use neutral dark/light surfaces with one distinctive accent. Final colours must pass accessibility contrast checks and must not imitate Perforce or OpenAI trade dress.

## Listing-copy draft

### Name
Perforce Companion

### Short description
Safely inspect Perforce workspaces, changelists, file locks, and UnrealGameSync status from ChatGPT.

### Longer description
Perforce Companion connects ChatGPT to an approved local Perforce workspace through a secure companion application. Review opened files, pending changelists, asset locks, file history, and UnrealGameSync build status without exposing unrestricted command or filesystem access. Write actions, when enabled, use previews and explicit confirmation.

## Positioning boundaries

Do not use:
- "Official Perforce app"
- "Built by Perforce"
- "Official ChatGPT integration"

Unless the applicable company has explicitly authorised that language.
